# F.R.I.D.A.Y — Android AI Assistant

A real Android AI assistant app powered by Gemini AI — like Google Assistant but yours!

## Features
- 🎙️ Voice commands (tap mic and speak)
- 🤖 Gemini AI for smart conversations
- 📱 Open any app by voice: "Open Instagram"
- 📞 Make calls: "Call John"
- 💬 Send WhatsApp: "WhatsApp Mum and say I'm coming home"
- 💬 Send SMS: "Text 0812345678 and say hello"
- ⏰ Set alarms: "Set alarm at 7 AM"
- ⏱️ Reminders: "Remind me to drink water in 30 minutes"
- 🔍 Search: "Search weather in Lagos"
- 📺 YouTube: "Play Afrobeats on YouTube"
- 🗺️ Maps: "Navigate to Ikeja"
- 📰 News, Weather, Battery level, and more!

## How to Build (FREE — using Android Studio)

### Step 1 — Install Android Studio
Download from: https://developer.android.com/studio
Install it on your PC/laptop (Windows, Mac, or Linux)

### Step 2 — Open the project
1. Open Android Studio
2. Click **Open** → select the `FridayAI` folder
3. Wait for Gradle to sync (2-5 minutes)

### Step 3 — Get your Gemini API key
1. Go to https://aistudio.google.com
2. Sign in with Google
3. Click **Get API Key** → **Create API key**
4. Copy the key (starts with AIza...)

### Step 4 — Build the APK
1. In Android Studio: **Build → Build Bundle(s) / APK(s) → Build APK(s)**
2. Wait ~2 minutes
3. Click **Locate** to find the APK file
4. Transfer the APK to your Android phone
5. Install it (allow "Install from unknown sources" in settings)

### Step 5 — Setup Friday
1. Open the app
2. Paste your Gemini API key
3. Enter your name
4. Tap **ACTIVATE FRIDAY**
5. Grant all permissions when asked
6. Start talking! 🎙️

|----------|-----------------|
| "Call John" | Calls John from contacts |

| "Text 08012345678 and say hello" | Sends SMS |
| "Set alarm at 6:30 AM" | Sets alarm |
| "Remind me to pray in 20 minutes" | Sets reminder |

| "Open Spotify" | Opens Spotify |
| "Play Davido on YouTube" | Searches YouTube |
| "Search weather today" | Googles weather |
| "Navigate to Victoria Island" | Opens Google Maps |
| "What's the time?" | Tells time |
| "How are you?" | Friday chats with you via Gemini |

## Set Friday as Default Assistant (like Google Assistant)
1. Go to **Settings → Apps → Default Apps → Digital Assistant**
2. Select **F.R.I.D.A.Y**
3. Now hold the home button to wake Friday!
